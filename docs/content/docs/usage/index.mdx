The engine is written in Rust, but you don't need to know Rust to use it. Two sets of bindings expose the engine to higher-level languages:

| Binding | Package | Install | Inference | Use case |
|---------|---------|---------|-----------|----------|
| **Python** | [`hexchess-zero`](https://pypi.org/project/hexchess-zero/) (PyPI) | `pip install hexchess-zero` | ONNX Runtime | Training, analysis, research |
| **TypeScript / WASM** | [`hexchess-zero`](https://www.npmjs.com/package/hexchess-zero) (npm) | `npm install hexchess-zero` | Tract (pure Rust) | Browser apps, play against AI |

Prebuilt wheels are published for Linux, macOS, and Windows (Python 3.9–3.13); the npm package ships the WASM binary and JS loader. The PyPI distribution is named `hexchess-zero` but imports as `hexchess`.

Both bindings expose the same core concepts:

- **`Game`** — a game state that you can query (legal moves, board state, status) and mutate (apply/undo moves)
- **AI search** — MCTS-based move selection with configurable strength (simulation count) and optional neural network model

## Move format

Moves are represented as typed `Move` objects with `from_q`, `from_r`, `to_q`, `to_r`, an optional `promotion` (`"queen"`, `"rook"`, `"bishop"`, or `"knight"`), and a `notation` string. The `(q, r)` pair is the engine's internal cell address; the `notation` field gives you the human-readable [Glinski notation](/docs/rules#notation) form (e.g. `"f5-f6"`, `"f10-f11=Q"`).

You can also apply moves directly by notation — `game.apply("f5-f6")` works in both Python and TypeScript.

## Game status

The `status()` method returns one of:
- `"ongoing"` — game is in progress
- `"checkmate_white"` — White wins by checkmate
- `"checkmate_black"` — Black wins by checkmate
- `"stalemate"` — draw by stalemate
- `"draw_repetition"` — draw by threefold repetition
- `"draw_fifty"` — draw by 50-move rule
- `"draw_material"` — draw by insufficient material

Choose your language to get started:

- [Python](/docs/usage/python) — for training, analysis, and scripting
- [TypeScript](/docs/usage/typescript) — for browser applications
